# Aula-18-frontend-autenticacao

Esse repositório é construído por cima do frontend presente na Aula-16-Frontend-Crud: [Repo-Aula-16](https://github.com/philcisco-classroom/aula-16-nextjs-crud).
O objetivo é adicionar as seguintes funcionalidades:

- Criação do Forms Login de Usuário
- Botão Logout
- Validação do Forms com ZOD
- Enviar cookie com Token para o backend

# Bibliotecas

Para o correto funcionamento do frontend precisamos instalar as seguintes bibliotecas (caso você apenas vá clonar esse projeto, elas já estão no package.json, basta executar $ npm i)

     - $ npm i sonner (toast com mensagens de erro)
     - $ npm i zod (biblioteca para validação do forms)

# Novos arquivos

# Criar .env

- NEXT_PUBLIC_API_URL="http://localhost:3001"

# Execução

 - $ npm i
 - $ npm run dev
